# 👩‍🏫 REST API for School application
>  🚀 Deployed at Heroku

## 🛠️ Technologies
The following tools are being used on the development of this API:
- NodeJS
- Nodemon
- Sucrase
- Express
- Sequelize
- PostgreSQL
- Docker
- bcryptjs
- JWT (jsonwebtoken)
- Multer
- Eslint
> See package.json for more details

## ✅ Features
- [x] User CRUD
- [x] User Authentication
- [x] Students CRUD
- [x] File uploads
- [x] Profile picture Storage
- [x] Students List
